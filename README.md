# Hellometer Public Website

The public-facing website for Hellometer, built with [Next.js](https://nextjs.org/) and [Tailwind CSS](https://tailwindcss.com/).

## Getting Started

```bash
cd hellometer-io
pnpm install
pnpm dev
```
